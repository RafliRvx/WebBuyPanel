# Website Transaksi Panel Pterodactyl

Website untuk jualan panel Pterodactyl otomatis dengan payment gateway Pakasir.

## Fitur
1. Landing page dengan informasi lengkap
2. Sistem login/register dengan validasi password
3. Pricing page dengan berbagai paket RAM
4. Pembayaran QRIS otomatis via Pakasir
5. Create panel otomatis di Pterodactyl
6. Dashboard user untuk melihat riwayat order
7. Dashboard admin untuk manage semua data
8. Notifikasi Telegram untuk setiap transaksi
9. Penyimpanan data di file JSON
10. Support deploy di localhost dan Vercel
